# Cyber Toolkit

Cyber Toolkit es una herramienta modular de redes y ciberseguridad desarrollada en Python.

El proyecto está diseñado para aprender networking, automatización y arquitectura de herramientas de ciberseguridad.

---

## Features

✔ Port Scanner  
✔ Ping Tool  
✔ DNS Lookup  
✔ HTTP Checker  
✔ Sistema de reportes automáticos  
✔ Arquitectura modular

---


## Instalación

Clona el repositorio:

```bash
git clone URL_DEL_REPO
```

Entrar en el proyecto:

```bash
cd cyber-toolkit
```

---

## Uso

Ejecutar el programa:

```bash
python main.py
```

---

## Herramientas

### Port Scanner

Escanea puertos abiertos en un host objetivo.

### Ping Tool

Comprueba si un host está activo.

### DNS Lookup

Resuelve dominios a direcciones IP.

### HTTP Checker

Comprueba si un host tiene el puerto HTTP abierto.

---

## Reportes

Los resultados se guardan automáticamente en la carpeta:

```text
reports/
```
